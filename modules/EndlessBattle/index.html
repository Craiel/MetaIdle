Updates
Beta Version 0.2 - 16/05/14
- Renamed the Thief's special ability upgrades to something more fitting
- The window will now resize automatically on any resolution
- Dragging and dropping items should now be bug free
- Your stat bonuses from leveling up now load properly, you have not lost any stats on current saves
- Recreated how particles are created and drawn
- Condensed a lot of the code
- Fixed a bug where the Thief's special ability upgrades would not increase by the right amount, all current saves will now load with the correct amount, you have not lost anything
- You can now unlock new attacks through the upgrades menu after meeting certain requirements
- You can now unlock auto selling upgrades
- Epic and Legendary items can now have special effects
- Added the first events, these occur randomly every hour
- Added new Quests that occur from events, these reward powerful buffs in addition to their other rewards
- All interface windows are now independant and can be dragged around the screen
- Fixed a bug where stats displayed in the Character window could overlap other stats
- Added a loading screen
- Added a new stat: Evasion
- Thieves have turned into Assassins
- Assassins grant a bonus to Evasion instead of armour
- Rejuvenating Strikes is now weaker in the early game but stronger in the late game
- Items and Monsters have been rebalanced, you should now be able to progress further faster
- Weapons of higher rarities will now always have higher damage than weapons of lower rarities
- Strength, Agility and Stamina have been changed. Crit chance should now be a more valuable stat on high level items
- The armour formula has been changed, it should no longer be an easy feat to reach maximum damage reduction
- The amount of stacks on a debuff is now displayed on the debuff's icon
- Because of the power of the new items, the spawn chance of bosses and drop rates of higher rarity items has been decreased
- The power of abilities has been reduced to make them more balanced with the new weapon damage
- You can no longer choose a bonus to Crit Chance when leveling up, this is to make Crit Chance on higher level items more appealing
- An alert will appear when a new quest or upgrade is available
- The quests button will now glow when there is a new quest
- Resetting the game will now grant you an amount of power shards depending on how well you did
- Added some ads under the game to help me keep the server running. If you don't like the ads you are free to use adblock, but please remember that these ads are needed to pay for the server

Beta Version 0.1b
- The critical strike particle now uses the correct particle
- Fixed a bug where damage bonuses from leveling up would not work
- Added a tooltip to the damage bonus stat in the character window
- The damage bonus upgrade option from leveling up no longer displays an incorrect tooltip
- Fixed a bug where the game would attempt to equip items that did not exist
- Fixed a bug where the monster damage particles option would not work correctly
- Fixed a bug where mercenary ability upgrades would expire
- Fixed a bug where gold gain would increase mercenary Gps by twice the amount
- Increased the amount of experience required to gain levels. Leveling felt too easy and and this should make Exp Gain a more valuable stat
- Sadly the mercenary ability upgrades and the damage bonuses from leveling up you have already obtained will not be recoverable. So I've added stat increases to health and hp5 when you level up. This WILL work from current saves so it's up to you if you want to reset. I'm really sorry about this.

Beta Version 0.1a
- Commanders now increase your Health and Thieves increase your Armour. Their gold gain abilities made them incredibly overpowered :( More gold gain features are planned though.
- Nerfed the Cleric, Mage and Warlock bonuses to 5%. 100% bonuses for just 10 mercenaries made them too powerful
- Gold Earned is now correctly formatted in the stats menu
- Added more tooltips to the stats in the equipment window
- Added a "Sell All" button to the inventory
- Repositioned the inventory so it doesn't cover up the experience bar
- Quest descriptions can now english better
- When you level up, some of the stat upgrades will now have tooltips
- The Upgrades Button will now glow when a new upgrade becomes available
- You can now alt-right-click to equip a trinket into the second slot
- Resetting will now reset the Resurrection interfaces
- Removed annoying console logs
- You can no longer drag inventory/equipment slots around the screen
- You can now right click an equipped item to unequip it
- Item tooltips now display the item's level, all current items did not have this feature and so will have an undefined item level
- Fixed a bug where stat bonuses from leveling up would not work properly. Sadly any bonuses you gained have been lost, you can either continue without them gaining new ones as you level, or reset your save. I'm sorry :(
- Fixed a bug where battle level buttons would not load in the correct state
- Your battle level will now save
- Hovering over a monster's health bar will now show their health
- Added an options menu:
- Control which battle particles show
- Toggle player health, monster health and experience to always show
- Items now have a coloured border depending on their rarity
- Fixed a bug where special upgrades would make normal upgrades unpurchasable
- Fixed a bug where items in your inventory would still be displayed after resetting
- Cleaned up the code a bit and removed a lot of unused graphics
